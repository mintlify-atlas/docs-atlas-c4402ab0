## Overview

StreamSwap provides auto-generated TypeScript bindings for type-safe contract interactions. The SDK includes:

- Type definitions for all contract messages
- Query and execute client classes
- Message composer utilities
- Full TypeScript type safety

## Schema Generation

Generate TypeScript bindings from contract schemas:

```bash
scripts/schema.sh
```

### Generation Process

The `schema.sh` script performs the following steps:

1. **Generate Schemas**: Run `cargo schema` for each contract
2. **TypeScript Codegen**: Generate TypeScript types from JSON schemas
3. **Cleanup**: Remove temporary schema files

```bash
#!/bin/bash

# Generate schemas for all contracts
for d in "$CONTRACTS_DIR"/*; do
  if [ -d "$d" ]; then
    cd "$d" || exit
    cargo schema
    cd "$MAIN_DIR" || exit
  fi
done

# Generate TypeScript types
cd "$TS_DIR" || exit
yarn install
yarn generate-ts

# Clean up schema files
for d in "$CONTRACTS_DIR"/*; do
  if [ -d "$d" ]; then
    rm -rf "$d"/schema
  fi
done
```

<Tip>
  Run `scripts/schema.sh` whenever you modify contract message types to regenerate TypeScript bindings.
</Tip>

## StreamSwapController Client

The controller client manages stream creation and protocol parameters.

### Query Client

Read-only access to contract state:

```typescript
import { StreamSwapControllerQueryClient } from './types/StreamSwapController.client';
import { CosmWasmClient } from '@cosmjs/cosmwasm-stargate';

// Initialize client
const client = await CosmWasmClient.connect('https://rpc.osmosis.zone');
const queryClient = new StreamSwapControllerQueryClient(
  client,
  'osmo17kqws7fy2qnejdw68cqjlw456gdxcdukk0wq6026u826zlda3y7qdhzqlv'
);

// Query protocol parameters
const params = await queryClient.params();
console.log('Exit fee:', params.exit_fee_percent);

// Get last stream ID
const lastId = await queryClient.lastStreamId();
console.log('Last stream ID:', lastId);

// List all streams
const streams = await queryClient.listStreams({
  limit: 10,
  startAfter: 0
});
console.log('Streams:', streams);
```

### Execute Client

For executing transactions:

```typescript
import { StreamSwapControllerClient } from './types/StreamSwapController.client';
import { SigningCosmWasmClient } from '@cosmjs/cosmwasm-stargate';
import { DirectSecp256k1HdWallet } from '@cosmjs/proto-signing';

// Setup signing client
const wallet = await DirectSecp256k1HdWallet.fromMnemonic(mnemonic);
const [account] = await wallet.getAccounts();
const client = await SigningCosmWasmClient.connectWithSigner(
  'https://rpc.osmosis.zone',
  wallet
);

// Initialize execute client
const executeClient = new StreamSwapControllerClient(
  client,
  account.address,
  'osmo17kqws7fy2qnejdw68cqjlw456gdxcdukk0wq6026u826zlda3y7qdhzqlv'
);

// Create a new stream
const result = await executeClient.createStream(
  {
    msg: {
      name: 'My Token Sale',
      treasury: account.address,
      out_denom: 'factory/osmo.../mytoken',
      in_denom: 'uosmo',
      start_time: '1704067200000000000',
      end_time: '1704153600000000000',
      // ... other parameters
    }
  },
  'auto',
  undefined,
  [{ denom: 'uosmo', amount: '100' }] // creation fee
);

console.log('Stream created:', result.transactionHash);
```

## StreamSwapStream Client

The stream client handles individual stream interactions.

### Query Operations

Access stream and position data:

```typescript
import { StreamSwapStreamQueryClient } from './types/StreamSwapStream.client';

const streamClient = new StreamSwapStreamQueryClient(
  client,
  'osmo1...' // stream contract address
);

// Get stream info
const stream = await streamClient.stream();
console.log('Stream status:', stream.status);
console.log('Out remaining:', stream.out_remaining);

// Get position
const position = await streamClient.position({
  owner: 'osmo1...'
});
console.log('Shares:', position.shares);
console.log('Purchased:', position.purchased);

// List all positions
const positions = await streamClient.listPositions({
  limit: 50,
  startAfter: undefined
});

// Get pricing data
const avgPrice = await streamClient.averagePrice();
const lastPrice = await streamClient.lastStreamedPrice();
```

### Execute Operations

Interact with streams:

```typescript
import { StreamSwapStreamClient } from './types/StreamSwapStream.client';

const streamClient = new StreamSwapStreamClient(
  client,
  account.address,
  'osmo1...' // stream contract address
);

// Subscribe to stream
const subResult = await streamClient.subscribe(
  'auto',
  undefined,
  [{ denom: 'uosmo', amount: '1000000' }] // subscription amount
);

// Withdraw unspent tokens
const withdrawResult = await streamClient.withdraw(
  { cap: '500000' }, // optional cap
  'auto'
);

// Exit stream after it ends
const exitResult = await streamClient.exitStream(
  { salt: undefined },
  'auto'
);

// Finalize stream (treasury only)
const finalizeResult = await streamClient.finalizeStream(
  {
    newTreasury: undefined,
    createPool: {
      pool_config: {
        // Pool configuration
      }
    },
    salt: undefined
  },
  'auto'
);
```
